My person [eslint](http://eslint.org) config. See [lib/index.js](lib/index.js) for more information on each rule.

Usage:

## .eslintrc

```json
{
  "extends": [
    "jongleberry"
  ]
}
```

## package.json

Run the following:

```bash
npm i --save-dev babel-eslint eslint-config-jongleberry eslint-plugin-react eslint-plugin-standard
```
